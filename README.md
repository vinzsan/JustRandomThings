1. **Instalasi NASM:**
   ```bash
   sudo apt install nasm
   ```
2. **Assemble Program:**
   ```bash
   nasm -f elf32 program.asm -o program.o
   ```
3. **Link the Object File:**
   ```bash
   ld -m elf_i386 -s -o program program.o
   ```
4. **Run**
   ```bash
   ./program_kalian
   ```
